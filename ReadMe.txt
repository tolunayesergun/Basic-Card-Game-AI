Pokemon.rar ı Bir klosöre çıkarttıktan sonra, netbeans programında içe aktarımı yapılır.
veya pokemon klosörünün içinde dist klosörüne girilip Pokemon.jar çalıştırılarak doğrudan oyuna girilir.
Oyun içinde main menuden oyun modunu seçebilirsiniz. 
Arena Ekranında Sol üst köşede Kartları göster butonuna basıp, Kapalı kartları gösterebilirsiniz.
Sağ üst köşedeki Ana Menu butonundan, Main Menuye geri dönebilirsiniz.
